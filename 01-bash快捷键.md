# Bash常用快捷键

```bash
CTRL + 左箭头			  # 每次向左移动一个单词
CTRL + 右箭头			  # 每次向右移动一个单词
CTRL + A		  		# 移动光标到开头
CTRL + E		  		# 移动光标到结尾
CTRL + U		  		# 删除光标前所有内容
CTRL + K		 		# 删除光标后所有内容
CTRL + W				# 删除光标之前到空格之间的内容
CTRL + L		  		# 清屏=clear
CTRL + C		  		# 终止当前命令的执行
CTRL + Z		  		# 暂停，放到后台
CTRL + D		  		# 注销=logout

CTRL + S		  		# 暂停输出到屏幕，锁屏，不显示内容
CTRL + Q		  		# 恢复输出到屏幕

ESC + .			  		# 调出上一个命令的最后一个空格后的内容
```
